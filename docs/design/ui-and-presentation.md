# UI & Presentation Design

> **Version**: 0.1.0  
> **Last Updated**: 2026-01-14

## Overview

RedGate's visual design embraces a **retro-fantasy aesthetic** combining 2-bit graphics with CRT filter effects. The UI is themed around a **Fantasy OS metaphor**, presenting the downtime mode as an operating system interface similar to Windows 95/98 or Kingsway.

## Visual Style

### Color Palette

The game uses a strict **4-color palette** (2-bit) with variations for different contexts.

#### Primary Palette (Default)

| Color | Hex | Usage |
|-------|-----|-------|
| **Background** | `#1a1a2e` | Base/darkest |
| **Shadow** | `#4a4a6a` | Mid-dark accents |
| **Highlight** | `#8a8aaa` | Mid-light accents |
| **Foreground** | `#eaeaea` | Text/brightest |

#### Alternative Palettes

Different areas or states can use shifted palettes:

**Dagsgard (Downtime)**

### CRT Filter

A shader-based CRT effect applied over the entire game:

```gdshader
shader_type canvas_item;

uniform float scanline_intensity : hint_range(0.0, 1.0) = 0.15;
uniform float curvature : hint_range(0.0, 0.5) = 0.1;
uniform float vignette_intensity : hint_range(0.0, 1.0) = 0.3;
uniform float chromatic_aberration : hint_range(0.0, 5.0) = 1.0;
uniform float noise_intensity : hint_range(0.0, 0.5) = 0.05;

// Scanlines
vec4 apply_scanlines(vec4 color, vec2 uv) {
    float scanline = sin(uv.y * 800.0) * scanline_intensity;
    return color * (1.0 - scanline);
}

// Barrel distortion for CRT curvature
vec2 apply_curvature(vec2 uv) {
    vec2 centered = uv * 2.0 - 1.0;
    vec2 offset = centered * dot(centered, centered) * curvature;
    return (centered + offset) * 0.5 + 0.5;
}
```

### Sprite Guidelines

#### Resolution
- **Tile Size**: 16x16 pixels base
- **Character Sprites**: 16x16 to 32x32
- **UI Elements**: 8-pixel grid alignment

#### Animation
- **Idle**: 2-4 frames, slow loop
- **Walk**: 4-6 frames
- **Attack**: 3-4 frames
- **Death**: 3-4 frames, fade out

#### Style Rules
- Maximum 4 colors per sprite (matching active palette)
- 1-pixel outlines for important elements
- Dithering patterns for gradients
- Consistent light source (top-left)

## Sound Design Guidelines

### Audio Categories

| Category | Description | Volume Default |
|----------|-------------|----------------|
| **Master** | Overall volume | 80% |
| **Music** | Background tracks | 60% |
| **SFX** | Game effects | 80% |
| **UI** | Interface sounds | 70% |
| **Ambient** | Environmental | 50% |

### UI Sounds

| Action | Sound Type |
|--------|------------|
| Button click | Short click/pop |
| Window open | Ascending chime |
| Window close | Descending chime |
| Error | Low buzz/boop |
| Success | High chime |
| Notification | Bell ding |
| Type character | Soft tap |
| Item pickup | Coin/pickup jingle |

### Combat Sounds

| Action | Sound Type |
|--------|------------|
| Hit (melee) | Impact thud |
| Hit (ranged) | Projectile impact |
| Critical hit | Enhanced impact + chime |
| Enemy death | Fade out + drop |
| Voice collect | Ethereal whoosh |
| Level up | Triumphant fanfare |
| Low health | Heartbeat pulse |

### Ambient Sounds

| Location | Sound Type |
|----------|------------|
| Dagsgard | Cave echoes, distant hammering |
| Forest | Wind, leaves, distant birds |
| Ruins | Wind, creaking, drips |
| Combat | Tension drone, escalating |

## Accessibility Considerations

### Color Blindness Support
- Use patterns in addition to color for status
- High contrast mode option
- Configurable palette options

### Screen Reader Support
- Alt text for UI elements
- Keyboard navigation for all menus
- Audio cues for state changes

### Motor Accessibility
- Auto-attack reduces input requirements
- Adjustable game speed option
- Hold-to-move option
- Rebindable controls

### Visual Accessibility
- Scalable UI elements
- CRT filter toggle (can disable)
- Screen shake toggle
- High visibility mode for enemies

## Resolution and Scaling

### Target Resolutions
- **Native**: 854x480 (16:9)
- **Scale**: Integer scaling to target resolution
- **Maximum**: 1920x1080 (2.25x scale)

### UI Scaling
```csharp
UIScale = min(ScreenWidth / 854, ScreenHeight / 480)
UIScale = floor(UIScale) // Integer scaling only
```

### Font Sizes
- **Small**: 8px (scaled)
- **Normal**: 12px (scaled)
- **Large**: 16px (scaled)
- **Title**: 24px (scaled)
